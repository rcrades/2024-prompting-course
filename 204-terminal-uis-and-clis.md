# Terminal UIs and CLIs: Why the Command Line is Back

> **Status**: Outline — full content coming soon

## The Big Idea

The terminal — that text-based interface developers have used for decades — is suddenly more relevant than ever. AI tools are making it accessible to newcomers, and some of the most powerful AI agents live there.

## What is the Terminal?

The terminal (also called the command line or CLI) is a way to interact with your computer using text commands instead of clicking buttons. You type a command, press enter, and get a result.

It might look intimidating at first, but here's the thing: **AI agents can help you use it.** You can literally ask Claude Code "how do I do X?" and it will run the commands for you.

## Why is it Having a Moment?

- **AI agents live here**: Claude Code, Codex, and many other AI tools run in the terminal
- **It's faster for many tasks**: Once you know the basics, text commands can be quicker than clicking through menus
- **TUIs (Terminal User Interfaces)**: A new wave of tools that combine the speed of the terminal with visual elements — the best of both worlds
- **Developer culture is shifting**: More tools are being built for the terminal because AI makes it more accessible

## What's a TUI?

A TUI (Terminal User Interface) is a program that runs in the terminal but has visual elements — menus, panels, colors, interactive lists. Think of it as a middle ground between a full graphical app and plain text commands.

Examples:
- **Claude Code** itself has a TUI — it shows you diffs, file trees, and progress
- **lazygit** — a visual way to use Git from the terminal
- **btop** — a beautiful system monitor
