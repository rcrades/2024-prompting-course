## 📖 What is This Course?

The ability to effectively communicate with AI tools has become a critical skill. This course teaches you to **think alongside AI** while building real web applications from scratch.

**You'll learn to:**
- 🎯 Master prompting techniques for technical problem-solving
- 💻 Build web applications using HTML, CSS, JavaScript, and Python
- 🤖 Leverage ChatGPT, Claude, and Cursor AI as development partners
- 🔍 Debug, iterate, and improve with AI assistance
- 🌐 Deploy your projects using GitHub

**Perfect for:**
- Folks that didn't have plans to learn how to code
- Anyone curious about AI-assisted development
- Self-learners wanting structured guidance
- Professionals upskilling for an AI-augmented future

## 🎓 Course Structure

### Week 0: Foundation
**Get started with AI tools and basic prompting**
- Set up ChatGPT, Claude, and GitHub accounts
- Learn fundamental prompting principles
- Build your first "About Me" webpage
- Master the art of asking AI for help

### Week 1: Fundamentals
**Dive into web technologies with AI guidance**
- Understand HTML, CSS, and JavaScript
- Use Cursor AI for code development
- Create a complete personal website
- Version control with GitHub

### Week 2: Intermediate
**Level up with responsive design and advanced prompting**
- Build mobile-friendly layouts
- Craft complex, multi-step prompts
- Develop an interactive web application
- Refine your AI collaboration workflow

### Week 3: Graduation
**Capstone project bringing it all together**
- Introduction to Python for backends
- Work with JSON data structures
- Build a House Issues Logging App
- Present your AI-assisted development journey

## 🎯 Learning Philosophy

### The Prompting-First Approach

This course teaches you to **communicate effectively with AI** from day one. You'll learn a structured prompting framework:

```
"I am trying to learn prompting and web development, but I am a beginner.
Can you help me with [specific task]?

If providing code, please use format 'languagename-section-sectionnumber-sectiontheme'
with comments and file specifications."
```

### Why This Matters

- **Future-proof skills**: AI tools are reshaping every industry
- **Accelerated learning**: Learn faster by asking better questions
- **Problem-solving mindset**: Understand how to break down complex tasks
- **Career advantage**: Stand out in an AI-augmented workforce
